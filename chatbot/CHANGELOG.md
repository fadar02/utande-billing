# Changelog

All notable changes to KaStart AI Assistant will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-07-17

### Added
- Business Suite for SMEs:
  - Invoice & Billing management
  - Customer Support tickets
  - Proposals & Quotes generation
  - HR Management (employees, leave, performance)
  - Business Reporting & Dashboard
- Multi-Agent System (CEO, Programmer, Accountant, Lawyer, etc.)
- AI Reasoning Engine (step-by-step thinking)
- Long-Term Memory system
- Knowledge Base (RAG)
- Security System (roles, API keys, audit logs)
- Plugin System with 6 built-in plugins
- Voice Assistant (with fallbacks)
- File Processing (PDF, images, data)
- Workflow Automation
- Predictive Intelligence
- Screen Understanding
- Emotion Detection
- AI Research Mode

### Changed
- Renamed from Nova to KaStart
- Improved routing for business commands
- Enhanced help system with business commands

### Fixed
- Invoice amount parsing with $ signs
- Support ticket ID matching with hyphens
- Leave request ID matching with hyphens
- Proposal/quote ID matching with hyphens
- Knowledge base "Add knowledge:" pattern matching
- Dashboard data loading (reports reload on each call)

- Initial release as Nova AI Assistant

- Recommendation engine
- Code generation (websites & mobile apps)
- File operations, code search, git, web fetch

- Knowledge base

- Plugin system
- Voice module
